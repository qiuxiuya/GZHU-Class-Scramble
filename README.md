# GZHU-Class-Scramble

广州大学自动选课脚本。

## 环境准备

1. 安装 Python 3.7+
2. 安装 Node.js
3. 安装依赖：

```bash
pip install -r requirements.txt
```

> 没有 Node.js 会报错 `RuntimeUnavailableError: Could not find an available JavaScript runtime`。

## 使用步骤

### 第一步：配置信息

运行 `python main.py`，输入 `0` 进入配置助手，按提示依次输入：

- 学号、密码
- 跳过容量检查的板块（直接回车默认 "通识"）
- 请求间隔（直接回车默认 0.5s）
- 选课模式：`1` 自动抢课 / `2` 捡漏 / `3` 替换
- 教学班号、板块、优先级

也可以在 [`配置信息.ini`](配置信息.ini) 里手动填写，格式如下：

```ini
[baseinfo]
username = 你的学号
password = 你的密码
starttime = 12:00          # 模式1定时启动，填 false 立即开始
skipcapacity = 通识         # 跳过容量检查的板块，逗号分隔
requestinterval = 0.5      # 请求间隔（秒）

[mode1]                    # 自动抢课：教学班号,板块,优先级
1 = 教学班号,主修课程,1

[mode2]                    # 捡漏模式：教学班号,板块
1 = 教学班号,通识选修

[mode3]                    # 替换模式：旧教学班号,新教学班号,板块
1 = 旧教学班号,新教学班号,主修课程
```

### 第二步：开始选课

配置好后，运行 `python main.py`，输入 `1` 登录教务系统，然后选择模式：

- **模式 1（自动抢课）**：到设定时间（默认 12:00）自动开抢。把 `starttime` 改成 `false` 则立即开始。
- **模式 2（捡漏）**：课程满员后持续监控，有空位自动选。
- **模式 3（替换）**：监控目标课程空位 → 自动退旧课 → 选新课。

## 注意事项

- 别开 VPN
- 别同时手动登教务系统，会被互相顶掉
- 别多开脚本
- 高峰期控制台可能没输出，但脚本在跑，等着就行
- 配置文件含密码，别发出去
